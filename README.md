## English

### Overview

BatteryChem-AI is an open-source AI-powered platform for high-throughput virtual screening of lithium-ion battery electrolyte formulations. Given a solvent + additive combination, it predicts **5 key electrochemical properties** simultaneously:

| Property | Physical Meaning | Unit |
|----------|-----------------|------|
| **Ionic Conductivity** | Fast-charging capability & rate performance | mS/cm |
| **Electrochemical Window** | High-voltage stability ceiling | V |
| **Flash Point** | Thermal runaway risk | °C |
| **Desolvation Barrier** | Li⁺ desolvation kinetics at the anode interface | kJ/mol |
| **Li⁺ Coordination Index** | SEI formation tendency & interfacial impedance | /100 |

The platform uses an ensemble of **RandomForest Regressors** trained on physics-based synthetic data, with **SHAP-based interpretability** and an interactive **Streamlit dashboard**.

### Architecture

```
Molecular Descriptors (15 solvents × 25 additives)
    → Data Generation Engine (56,250 synthetic samples)
    → RF Ensemble (5 models × 150 trees each)
    → 5-Fold Cross-Validation + SHAP Explainability
    → CLI / Streamlit Web Dashboard
```

### Features

- **Multi-objective prediction**: 5 properties in one shot with Pareto radar chart
- **High-throughput screening**: Vectorized screening of 10,800+ formulation combinations, Top 100 ranked
- **SHAP interpretability**: Feature importance attribution per prediction
- **Dual input modes**: Database presets (beginner-friendly) or custom molecular parameters
- **No external API**: Fully local computation, data sovereignty guaranteed
- **Interactive web UI**: Streamlit dashboard with real-time radar charts

### Data Generation Logic

The training dataset (56,250 samples) is generated by combining:
- **Base descriptors**: Hard-coded molecular properties from 15 industrial solvents and 25 functional additives
- **Gaussian noise perturbation**: 150 iterations of additive noise to expand the chemical space
- **Physical constraint models**: Empirical equations for ionic conductivity, electrochemical window, flash point, etc.

> ⚠️ **Important**: The training data is physics-based synthetic data, not real experimental measurements. Predictions should be treated as **virtual screening guidance**, not ground-truth values. We strongly recommend cross-validating predictions against literature experimental data before wet-lab validation. See [Validation Guide](#validation-guide) for details.

### Validation Guide

To improve prediction reliability, we recommend:

1. **Literature cross-validation**: Compare predictions against experimental data from journals (J. Electrochem. Soc., Energy Environ. Sci., etc.)
2. **Baseline comparison**: Use the built-in benchmark (EC:DMC 1:1 + 10 wt% FEC) as reference
3. **Confidence intervals**: Check if your candidate falls within the training distribution
4. **Physical sanity checks**: Predictions outside the training range (~0.1–17.5 mS/cm for conductivity) should be interpreted with caution

### 项目简介

BatteryChem-AI 是一个开源的锂离子电池电解液高通量虚拟筛选平台。给定溶剂 + 添加剂组合，可同时预测 **5 项关键电化学性质**：

| 性质 | 物理含义 | 单位 |
|------|---------|------|
| **离子电导率** | 快充能力与倍率性能 | mS/cm |
| **电化学窗口** | 高压稳定性上限 | V |
| **闪点** | 热失控风险 | °C |
| **去溶剂化能垒** | 锂离子脱溶剂化动力学阻力 | kJ/mol |
| **Li⁺ 配位指数** | SEI 形成倾向与界面阻抗 | /100 |

平台基于 **随机森林集成模型**，结合 **SHAP 可解释性分析**，提供交互式 **Streamlit 网页仪表盘**。
